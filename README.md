# solve-baekjoon
[Baekjoon Online Judge](https://www.acmicpc.net/) 사이트의 알고리즘 문제를 해결한 내용을 기록하는 저장소 입니다.

## 사용 언어
- JAVA

## 커밋 컨벤션
```
solve: 문제티어.문제번호 문제명

tag
- 문제태그
```
민문리 스터디에서 정한 문제일 경우 `solve(mml)`
